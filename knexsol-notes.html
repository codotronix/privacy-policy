Privacy Policy

Suman Barick built Knexsol Notes as a free note taking app. This SERVICE is provided by Suman Barick at no cost and is intended for use as is.

Knexsol Notes (hereby referred to as "this app") does not collect or store any user data. Please find below a detailed summary of all the data this app uses / requires to operate.

Detailed Summary of Data Collection and Usage

  • This app does not store or colect any user identifiable or private or public data. The notes / texts user creates are stored locally in user's device only and only shared when and to whom user wants to share with using native sharing widgets.
  • In future this app may provide users the ability to store the notes/texts they create to store on server but they will need to be authenticated and logged in first. Thus only users themselves will be able to see their notes and no other party can access whatever notes / texts they store in their personal logged in account.
